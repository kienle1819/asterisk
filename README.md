## How to install on Centos 7.
Run ```scripts-ast16.sh``` for build asterisk 16.
```
vi scripts-ast16.sh
chmod +x scripts-ast16.sh
./scripts-ast16.sh
```
## How to install on Ubuntu 18.04 & Debian 10,11
Run ```scripts-ast16.sh``` for build asterisk 20
```
nano scripts-ast16.sh
chmod +x scripts-ast16.sh
./scripts-ast16.sh
```
